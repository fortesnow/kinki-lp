# 家づくり相談所 LP

家づくり相談所のランディングページです。

## ファイル構成

- `index.html` - メインのHTMLファイル
- `styles.css` - スタイルシート
- `images/` - 画像ファイル（WebP形式）

## 使用方法

1. `index.html`をブラウザで開く
2. または、ローカルサーバーを起動して表示

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

## 注意事項

- スマートフォン向けに最適化されています
- PCでの表示は開発中は可能ですが、本番環境では非表示になります
